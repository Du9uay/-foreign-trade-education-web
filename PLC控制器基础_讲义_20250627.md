一、课程内容概述

1. [自动化行业基本构成]
自动化行业分为传感层、控制层（PLC）、执行层、管理层（SCADA/MES），核心硬件包括PLC、传感器、执行器、HMI、工业网络设备，典型应用有离散制造、过程控制、基础设施等场景。

2. [PLC的本质与特性]
PLC是工业控制器，具有一定的实时性，通过扫描周期来处理输入输出等操作，其硬件最小系统由CPU、电源、I/O模块组成，不同模块有不同功能。

3. [I/O接线规范]
I/O接线需遵循规范，两线制传感器和三线制传感器接线方式不同，同时要注意熔断器选型与继电器隔离等要求。

4. [梯形图五大基础指令]
梯形图的五大基础指令包括常开触点、常闭触点、输出线圈、TON定时器、计数器，这些是梯形图编程的基础元素。

5. [TIA Portal基础操作]
TIA Portal可进行工程创建、梯形图编写、程序下载监控等操作，是进行PLC程序开发调试的常用工具。

6. [急停的梯形图禁忌]
在梯形图编程中，急停电路有禁忌，不能出现双线圈冲突等情况，要遵循安全设计优先原则来进行急停相关的梯形图编程。

四、课程内容详解

### 一、行业认知与基础框架

#### （一）自动化行业全景图
1. **产业层级划分**  
   - **传感层**：是自动化系统获取信息的起始环节。例如在汽车装配线上，光电传感器用于检测零件位置，接近开关可感应金属物体的接近，它们将物理量（如位置、尺寸等）转化为电信号。  
   - **控制层（PLC）**：如同系统的“大脑”，接收传感层传来的电信号，依据预先编写的程序进行运算处理，再输出控制信号至执行层。以汽车装配线为例，PLC根据传感器检测到的零件位置信号，控制机械手按预定动作序列工作。  
   - **执行层**：将PLC的控制信号转化为实际动作。常见执行器有电机、电磁阀等。在汽车装配线中，机械手是执行器，依据PLC信号完成抓取、放置等动作；化工厂反应罐里，调节温度的加热装置、调节液位的阀门等属于执行器。  
   - **管理层（SCADA/MES）**：SCADA（数据采集与监视控制系统）和MES（制造执行系统）用于系统管理与监控。在智能楼宇中，SCADA系统可管理电梯调度、照明系统运行；MES系统能采集分析楼宇设备运行数据，实现智能化管理。  

2. **核心硬件构成**  
   - **PLC**：可编程逻辑控制器，是控制层核心设备。含CPU模块（运算处理）、输入输出（I/O）模块（连接传感器与执行器等外部设备）。  
   - **传感器**：如光电传感器、接近开关等，将物理量转化为电信号传递给PLC。例如温度传感器检测反应罐内温度，转化为电信号输入PLC。  
   - **执行器**：包括电机（电能转机械能，实现旋转或直线运动）、电磁阀（控制气体或液体通断）等。汽车喷漆车间中，电磁阀控制喷漆开关，电机带动喷漆设备移动。  
   - **HMI（人机界面）**：以触摸屏为例，操作人员可通过它设置参数、监控设备运行状态。智能楼宇电梯控制系统中，HMI可显示电梯运行楼层、等待人数等信息。  
   - **工业网络设备**：用于连接各设备实现数据传输，常见工业以太网交换机，可让PLC、传感器等设备通信，保障数据流畅。如大型工厂自动化系统中，工业以太网交换机连接各车间PLC与设备，实现数据实时传输共享。  

3. **典型应用场景认知**  
   - **离散制造（汽车装配线案例）**  
     汽车装配线有车身焊接、内饰安装等多道工序。在车身焊接工序，传感器检测到车身到达指定位置，将信号传至PLC，PLC依程序控制机械手各关节运动，完成焊接动作。此过程涉及PLC接收输入信号、内部程序处理、输出控制信号给执行器（机械手电机等），实现精确焊接。  
   - **过程控制（化工厂反应罐案例）**  
     化工厂反应罐需控制温度与液位。温度传感器实时检测罐内温度，将信号输入PLC，PLC据设定温度范围，控制加热或冷却装置的执行器调节温度；液位传感器检测液位高度，PLC依液位情况控制阀门等执行器调节液位，保证工艺参数正常。  
   - **基础设施（智能楼宇案例）**  
     智能楼宇电梯调度系统中，PLC管理电梯运行。乘客在电梯外按楼层按钮，传感器检测到信号传至PLC，PLC依各电梯位置、负载等情况，调度合适电梯接客。照明系统中，PLC根据时间、光线强度等因素控制灯光开关与亮度，如白天光线充足时，自动关闭部分公共区域灯光以节能。  


#### （二）核心能力需求
1. **硬件能力**  
   - **看懂电气图**：电气图是系统蓝图，需识别电气元件符号与连接方式。例如汽车装配线电气图中，要能识别PLC的I/O接口、传感器接线端子、执行器连接线路等。  
   - **选型PLC模块**：根据应用需求选合适PLC模块。估算DI/DO点数原则：统计系统需检测的输入信号数与需控制的输出信号数。如简单物料搬运系统，需检测4个输入信号、控制4个输出信号，则可选DI/DO点数至少为4/4的PLC模块。  
   - **安全接线**：急停电路需独立硬接线设计。急停按钮直接连PLC安全输入端口，电路单独布线，保障紧急情况下快速切断电源，确保人员设备安全。  

2. **软件能力**  
   - **梯形图编程**：梯形图是PLC常用图形化编程语言，含五大基础指令：  
     - 常开触点：用于接收启动等信号输入。  
     - 常闭触点：用于接收停止等信号输入。  
     - 输出线圈：控制执行器动作。  
     - TON定时器：用于延时控制，如延时启动设备。  
     - 计数器：用于计数相关操作。例如编写起保停电路，常开触点接启动信号，常闭触点接停止信号，输出线圈控制执行器。  
   - **故障诊断**：系统故障时，通过查看PLC状态指示灯、分析梯形图程序等诊断。如PLC某输出指示灯不亮，先检查对应输出线圈有无信号，再查输出模块、传感器是否正常。  

3. **系统思维**  
   - **信号流分析（输入→处理→输出）**：理解信号流动过程。以简单灯光控制系统为例，光线传感器检测光线强度为输入信号，PLC接收后处理，若光线强度低于设定值，输出信号控制灯光开启，即输入 - 处理 - 输出过程。  
   - **安全设计优先原则**：设计系统时始终把安全放首位。如电梯控制系统设计，除正常调度程序外，需考虑急停按钮设计，确保紧急时立即停止电梯运行，保障乘客安全。

### 二、PLC核心定位与硬件基础

#### （一）PLC本质与特性

##### 1. 工业控制器作用
PLC（Programmable Logic Controller）是工业自动化领域的核心设备，本质是为工业环境设计的数字运算电子系统。其作用是通过编程实现对工业生产过程中设备的逻辑控制、顺序控制、定时、计数和算术运算等。例如，在汽车装配线中，PLC可精准控制机械手按预设程序执行动作序列，像抓取、搬运、安装零件等操作，大幅提高装配效率与精度。


##### 2. 扫描周期与实时性
- **扫描周期**  
  PLC采用循环扫描工作方式，分为三个阶段：  
  - **输入采样阶段**：以扫描方式依次读取所有输入端子状态，存入输入映像寄存器。例如，汽车装配线中检测零件到位的传感器信号会在该阶段被读取。  
  - **程序执行阶段**：按先左后右、先上后下顺序扫描用户程序，进行逻辑运算并将结果存入输出映像寄存器。比如根据零件位置信号判断机械手动作程序。  
  - **输出刷新阶段**：将输出映像寄存器中输出继电器状态转存到输出锁存器，驱动外部负载。小型PLC扫描周期通常在几毫秒到几十毫秒，能满足多数工业控制需求。  
- **实时性**  
  实时性指PLC能及时响应输入信号变化并在规定时间完成输出操作。以化工厂反应罐温度控制为例，温度传感器检测到温度超设定值时，PLC需快速通过调节加热/冷却设备稳定温度，依赖扫描周期实现快速输入检测、程序执行与输出响应。


##### 3. 硬件最小系统组成
PLC硬件最小系统由CPU、电源、I/O模块构成：  
- **CPU模块**：核心部件，类似大脑，执行用户程序、处理数据、通信等。如解析梯形图程序，根据输入信号状态进行逻辑运算并控制输出状态。  
- **电源模块**：提供稳定工作电源，有直流（如24V）和交流（如220V）两种类型，为CPU、I/O模块等供电。  
- **I/O模块**：  
  - **输入（DI）模块**：接收外部设备输入信号，转换为PLC可识别数字信号。例如汽车装配线中零件到位检测传感器信号通过DI模块传入PLC。  
  - **输出（DO）模块**：输出PLC内部信号控制外部设备动作，转换为外部设备可接受信号。如PLC通过DO模块控制机械手电机实现抓取动作。  


##### 4. DI/DO点数估算原则
- **DI点数估算**：先确定输入信号点数量，如简单物料搬运系统有物料到位检测传感器（2个）、启动按钮（1个）、停止按钮（1个），初步DI点数为2+1+1=4个，再预留10%-20%余量应对后续功能扩展。  
- **DO点数估算**：确定输出设备数量，如物料搬运系统中控制电机正反转（2个输出点）、控制电磁阀（1个输出点），初步DO点数为2+1=3个，同样预留余量。  


##### 5. 安全规范基础
- **急停电路独立硬接线设计**：急停按钮采用独立硬接线连接PLC输入模块。例如急停按钮常闭触点直接连PLC输入点，按下急停按钮时，常闭触点断开，PLC立即触发停止相关设备程序，保障设备与人身安全，急停电路需独立于PLC程序控制，防止程序故障导致急停失效。

### 三、电气接线与信号认知  
#### （一）I/O接线规范  
##### 1. 两线制传感器接线  
- **概念介绍**：两线制传感器是指电源正负极共用两根线传输信号，常见工作电压为24V，例如24V两线制接近开关。  
- **接线方法**：  
  - 以24V两线制接近开关为例，将电源正极接到接近开关一端，接近开关另一端通过PLC输入端子连接至电源负极。  
  - **操作细节**：接线时需确保导线连接牢固，避免松动导致接触不良。  
  - **项目案例**：汽车装配线中，用两线制接近开关检测物料是否到位，按上述方法将接近开关连接至PLC输入点，实现物料到位信号的采集。  

##### 2. 三线制传感器接线  
- **概念介绍**：三线制传感器有电源正极、负极和信号输出线三根线，常见类型为NPN型和PNP型。  
- **接线方法**：  
  - **NPN型**：电源正极接接近开关VCC端，电源负极接GND端，信号输出线连接PLC输入端子。  
  - **PNP型**：电源正极接VCC，信号输出线连接输入端子，电源负极接GND。  
  - **操作细节**：需区分NPN与PNP型传感器的接线差异，避免接反。  
  - **项目案例**：智能楼宇电梯楼层检测中，使用三线制传感器，按NPN或PNP接线方式连接至PLC输入点，实现楼层信号采集。  


#### （二）熔断器选型  
##### 1. 选型原则  
- 熔断器额定电流应略大于电路正常工作电流。例如，某电路正常工作电流为5A，可选额定电流6A或7A的熔断器。  
- **操作方法**：  
  1. 测量电路正常工作电流。  
  2. 根据测量结果选择匹配额定电流的熔断器，且熔断器电压等级需与电路电压相匹配。  


#### （三）继电器隔离  
##### 1. 继电器隔离作用  
继电器隔离可防止外部电路干扰影响PLC正常工作，同时保护PLC输入输出端子。  
##### 2. 接线方法  
- 在PLC输入输出电路中接入继电器，外部信号先连接至继电器线圈，继电器触点连接PLC端子。  
- **操作细节**：外部强电信号需通过继电器隔离后连接PLC输入，避免强电损坏PLC。  
- **项目案例**：化工厂反应罐温度控制中，外部高温检测信号通过继电器隔离后连接PLC输入，保障PLC安全运行。

### 四、梯形图编程
#### （一）五大基础指令详解
##### 1. 常开触点
- **概念阐释**  
  常开触点是梯形图中模拟实际电路中常开开关的元素。其工作原理为：当对应的输入条件满足时，触点处于闭合状态，电流能够顺利通过；若输入条件不满足，触点保持断开，电流无法导通。例如，在一个简单的灯控电路中，若用常开触点来控制灯的开启，当连接到该触点的传感器检测到光线暗到一定程度时（输入条件满足），触点闭合，灯得电点亮。  
- **符号与操作**  
  梯形图中常开触点的符号为“┏━━━━━━━━┓”。在编程软件中插入常开触点的操作步骤：打开编程软件，进入梯形图编辑界面，找到触点插入功能，选择常开触点类型，然后关联对应的输入点地址（如PLC的输入点I0.0等）。

##### 2. 常闭触点
- **概念阐释**  
  常闭触点与常开触点相反，它模拟实际电路中的常闭开关。当对应的输入条件满足时，触点处于断开状态，电流无法通过；当输入条件不满足时，触点闭合，电流可以导通。比如，在电机过载保护电路中，常闭触点串联在电机控制回路中，当电机过载时（输入条件满足），常闭触点断开，切断电机电源，起到保护作用。  
- **符号与操作**  
  常闭触点在梯形图中的符号是“┏━━╋━━━━━━━━┓”。在编程软件中插入常闭触点的操作：进入梯形图编辑界面，找到触点插入功能，选择常闭触点类型，关联对应的输入点地址（如I0.1等）。

##### 3. 输出线圈
- **概念阐释**  
  输出线圈用于控制外部执行设备的动作。其工作机制为：当输出线圈所在的逻辑电路满足导通条件时，线圈得电，进而驱动外部的执行器（如电机、电磁阀等）工作。例如，在智能楼宇的照明控制中，输出线圈连接到照明灯具的控制电路，当通过前面的触点逻辑判断满足开灯条件时，输出线圈得电，点亮灯具。  
- **符号与操作**  
  输出线圈在梯形图中的符号为“┏━━━━━━━━┓○”。在编程软件中插入输出线圈的操作：打开梯形图编辑界面，找到输出线圈插入功能，选择对应的输出点地址（如PLC的输出点Q0.0等）进行关联。

##### 4. TON定时器（通电延时定时器）
- **概念阐释**  
  TON定时器实现通电延时功能。其工作原理是：当使能输入接通后，定时器开始计时，经过预设的时间延迟后，输出触点动作。例如，在化工厂反应罐温度控制中，当温度达到需要开启冷却设备的条件时，TON定时器使能输入接通，经过10秒预设时间后，输出触点闭合，启动冷却风扇。  
- **符号与操作**  
  TON定时器在梯形图中的符号包含使能输入（EN）、复位输入（R）、预设时间值（PT）等部分。操作步骤：在编程软件中找到TON定时器指令，设置复位输入（可连接手动复位信号）、使能输入（连接触发计时的触点）和预设时间值（如T#10S表示10秒），并关联对应的地址。

##### 5. 计数器
- **概念阐释**  
  计数器用于统计输入信号的脉冲次数，达到预设的计数次数后，输出触点动作。例如，在汽车装配线中，每装配一辆车会产生一个计数脉冲，使用计数器统计装配数量，当计数达到100辆时，输出触点动作，停止装配流程。  
- **符号与操作**  
  增计数器在梯形图中的符号包含计数输入（CU）、复位输入（R）、预设计数值（PV）等部分。操作方法：在编程软件中找到计数器指令，设置复位输入（连接清零信号）、计数输入（连接装配完成的脉冲信号）和预设计数值（如C#100表示预设计数100次），并关联对应的地址。

#### （二）编程范式示例——起保停电路
- **概念阐释**  
  起保停电路是梯形图中常用的基本控制电路，用于实现设备的启动、保持和停止功能。其逻辑为：启动按钮对应的常开触点闭合使线圈得电，保持触点（自身常开触点）保持线圈得电状态，停止按钮对应的常闭触点断开使线圈失电。  
- **项目案例与操作**  
  以电机起保停控制为例：首先，将启动按钮对应的常开触点连接到输出线圈的使能端，停止按钮对应的常闭触点串联在电路中。梯形图绘制步骤：插入启动按钮对应的常开触点（关联输入点I0.2）、停止按钮对应的常闭触点（关联输入点I0.3），然后插入输出线圈关联输出点Q0.1。当按下启动按钮时，I0.2常开触点闭合，Q0.1线圈得电，同时自身常开触点闭合实现保持；按下停止按钮时，I0.3常闭触点断开，Q0.1线圈失电，电机停止。

#### （三）电气与机械互锁实现
- **概念阐释**  
  电气互锁是通过电气元件的触点实现对电路的互锁控制，机械互锁是通过机械结构实现的互锁。例如，在电机正反转控制中，需要进行电气互锁和机械互锁。电气互锁通过正转接触器的常闭触点串联在反转控制电路中，反转接触器的常闭触点串联在正转控制电路中；机械互锁通过机械机构防止正反转接触器同时得电。  
- **操作示例**  
  在PLC编程中实现电机正反转电气互锁：插入正转启动按钮常开触点（关联I0.4）、反转启动按钮常闭触点（关联I0.5），正转接触器输出线圈Q0.2，反转接触器输出线圈Q0.3。梯形图中，Q0.2的常闭触点串联在Q0.3的控制电路中，Q0.3的常闭触点串联在Q0.2的控制电路中。这样当正转接触器得电时，其常闭触点断开，切断反转接触器的控制电路，反之亦然，实现电气互锁。

### 六、安全强化模块——急停的梯形图禁忌


#### （一）急停电路的重要性与基础认知
急停电路是工业控制系统中保障人员和设备安全的关键环节。例如在汽车焊接生产线中，当焊接机器人出现异常运动趋势时，急停电路需立即动作，使机器人停止运动，避免造成人员伤害或设备损坏。急停电路的核心作用是在紧急情况下快速切断设备的动力输出。


### （二）梯形图中急停的禁忌1：双线圈冲突
1. **双线圈冲突的概念**  
在梯形图编程中，**双线圈冲突指同一个输出线圈在梯形图中被多次独立赋值**。例如，有一个用于控制急停输出的线圈Q0.0，若在梯形图的不同网络中均对Q0.0进行了“= Q0.0”的操作，就会发生双线圈冲突。

2. **双线圈冲突的后果**  
由于PLC执行梯形图是按扫描周期顺序执行，最终**最后一次对该线圈的赋值会覆盖之前的赋值**。以如下错误程序为例：  
```
网络1
LD I0.0    // 急停按钮输入
O Q0.0     // 第一次对Q0.0赋值
= Q0.0

网络2
LD I0.1    // 其他逻辑输入
O Q0.0     // 第二次对Q0.0赋值
= Q0.0
```  
此时，Q0.0的最终状态由网络2的逻辑决定，而非由急停按钮I0.0的状态决定，导致急停功能失效。

3. **避免双线圈冲突的操作方法**  
通过中间继电器过渡来避免双线圈冲突。例如，先用中间继电器M0.0承载急停相关逻辑，再用M0.0控制输出线圈Q0.0，修改后程序如下：  
```
网络1
LD I0.0    // 急停按钮输入
= M0.0     // 用中间继电器M0.0存储急停状态

网络2
LD M0.0    // 调用中间继电器M0.0的状态
= Q0.0     // 控制急停输出线圈Q0.0
```  


### （三）急停梯形图的其他禁忌
1. **急停电路与其他电路的隔离禁忌**  
急停电路必须独立设计，不能与普通控制电路混合布线。例如，急停电路的输入输出线不能与模拟量信号或普通数字量信号线共用同一电缆，否则可能因干扰导致急停功能异常。在实际项目中，需专门铺设急停电路的独立线路，确保信号传输可靠。

2. **急停按钮接线禁忌**  
急停按钮通常采用**常闭触点**。若接线时误接成常开触点，会导致正常情况下急停按钮断开（设备可运行），而按下急停按钮时电路导通（无法实现急停）。梯形图编程时，需根据急停按钮的常闭特性编程，例如急停按钮接输入点I0.0，梯形图中应使用常闭触点：  
```
LD NOT I0.0  // 利用急停按钮常闭特性，按钮按下时I0.0为OFF，NOT I0.0为ON
...           // 后续急停控制逻辑
```  


### （四）项目案例：汽车装配线急停电路梯形图修正
以汽车装配线中机械手的急停控制为例，原错误程序存在双线圈冲突：  
```
// 错误程序片段
网络1
LD 急停按钮I0.0
O 机械手输出Q0.0
= Q0.0

网络2
LD 其他安全信号I0.1
O 机械手输出Q0.0
= Q0.0
```  
修正后程序：  
```
// 修正程序片段
网络1
LD 急停按钮I0.0
= 中间继电器M0.0  // 用中间继电器M0.0存储急停状态

网络2
LD M0.0
O 其他安全信号I0.1
= 机械手输出Q0.0  // 由中间继电器M0.0和其他安全信号共同控制机械手输出
```  

通过上述修正，避免了双线圈冲突，确保急停电路在紧急情况下能可靠动作，保障生产安全。

同学们需牢记急停梯形图的禁忌，在编程时严格遵循安全设计优先原则，确保工业控制系统的安全性与可靠性。
